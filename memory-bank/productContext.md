# Product Context

## Warum existiert dieses Projekt?

**macario.dev** ist die zentrale Business-Plattform für Lars Macario als No/Low-Code Entwickler, spezialisiert auf die Fitness- und Gesundheitsbranche. Die Website positioniert sich als Lösung für Personal Trainer, Fitnessstudios und Gesundheits-Coaches, die ihre Business-Prozesse digitalisieren möchten.

### Probleme die gelöst werden

1. **Lead-Generierung**: Strukturierte Erfassung von Projektanfragen aus der Fitness-Branche
2. **Projekt-Präsentation**: Professionelle Darstellung von Fähigkeiten und Referenzen speziell für Fitness-Professionals
3. **Kunden-Onboarding**: Standardisierter Prozess für neue Kunden
4. **Terminbuchung**: Einfache Buchung von Strategie-Gesprächen
5. **Credibility**: Professionelle Online-Präsenz mit authentischer Trainer-Story

## Wie sollte es funktionieren?

### User Journey

1. **Besucher** landet auf der Website
2. **Hero Section** präsentiert Haupt-CTA: "Kostenloses Strategie-Gespräch buchen"
3. **ProjectWizard** sammelt Projektinformationen:
   - Projekttyp (Trainer-Website, Buchungs-App, Kunden-Portal, Studio-Management, KI-Assistent, Automatisierung)
   - Verfügbar als Modal auf der Hauptseite und als Standalone-Landingpage unter `/wizard`
   - Budget
   - Zeitrahmen
   - Priorität
   - Projektbeschreibung (Fitness-spezifisch)
   - Kontaktdaten
4. **Projektanalyse** via OpenAI API
5. **Terminbuchung** für Strategie-Gespräch
6. **Kunden-Onboarding** für neue Kunden

### Zielgruppen

**Primäre Zielgruppen:**
- **Personal Trainer**: Selbstständige Trainer, die ihre Business-Prozesse digitalisieren möchten
- **Fitnessstudio-Betreiber**: Kleine bis mittelgroße Studios (1-5 Standorte)
- **Gesundheits-Coaches**: Ernährungsberater, Mental Health Coaches, Wellness-Experten
- **Physio-/Reha-Praxen**: Physiotherapeuten, Reha-Einrichtungen
- **Fitness-Influencer**: Content Creator im Fitness-Bereich

**Sekundäre Zielgruppen:**
- Yoga-/Pilates-Studios
- CrossFit-Boxen
- Online-Fitness-Coaches (rein digital)
- Supplement-/Sportnahrung-Unternehmen
- Sportvereine

**USP**: Lars Macario kombiniert 10+ Jahre Erfahrung als Personal Trainer mit No-Code Entwicklung – authentische Branchenkenntnis trifft auf moderne Technologie.

## User Experience Goals

- **Einfachheit**: Klare CTAs und einfache Navigation
- **Professionell**: Moderne, ansprechende UI
- **Vertrauen**: Portfolio und Referenzen zeigen
- **Effizienz**: Schnelle Projektanfrage ohne Barrieren
- **Transparenz**: Klare Preise und Prozesse
